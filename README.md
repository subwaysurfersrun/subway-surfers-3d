# 🚆 Subway Surfers 3D - Endless Runner

A fast-paced 3D endless runner game inspired by Subway Surfers, built entirely with HTML, CSS, JavaScript, and Three.js.

## 🎮 Features

- 🌍 Multiple world locations
  - New York
  - Rio
  - Tokyo
  - Paris
  - Cairo
  - Seoul

- 🏃 4 Playable Characters
  - Jake
  - Tricky
  - Fresh
  - Yutani

- 🛒 In-Game Shop System
  - Character customization
  - Unlockable items
  - Equipment system

- 🏆 Leaderboard System
  - All Time Rankings
  - Weekly Rankings
  - Daily Rankings
  - Player Search

- 🎁 Daily Rewards
  - Daily login bonuses
  - Reward streak progression

- 🎯 Mission System
  - Coin Collection
  - Distance Challenges
  - Power-Up Objectives

- ⚡ Power-Ups
  - Jetpack
  - Coin Magnet
  - Score Multiplier
  - Speed Boost

- 🎨 Fully 3D Graphics
  - Built using Three.js
  - Dynamic lighting
  - Animated characters
  - Responsive design

## 🕹 Controls

### Desktop

| Key | Action |
|------|---------|
| A / ← | Move Left |
| D / → | Move Right |
| W / ↑ / Space | Jump |
| S / ↓ | Slide |

### Mobile

| Gesture | Action |
|----------|---------|
| Swipe Left | Move Left |
| Swipe Right | Move Right |
| Swipe Up | Jump |
| Swipe Down | Slide |

## 🚀 Getting Started

### Clone Repository

```bash
git clone https://github.com/yourusername/subway-surfers-3d.git
```

### Open Project

```bash
cd subway-surfers-3d
```

Simply open:

```bash
index.html
```

in your browser.

No build process required.
